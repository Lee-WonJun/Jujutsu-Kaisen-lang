# 주술회전 프로그래밍 언어 (스칼라 구현체)

고죠 사토루 네놈은 JVM 마저 최강인 것이냐!!

## 실행

`scala`에 `project.scala` 가 있는 폴더를 기준으로 실행하면 됩니다.

```sh
# 실행
sclaa JJKlang-scala
sclaa --watch JJKlang-scala 

# 테스트
scala test JJKlang-scala
scala test  --watch JJKlang-scala
```

## 개발

```sh
# Metals LSP 환경 구축
cd JJKlang-scala
scala setup-ide .
```
